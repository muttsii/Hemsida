Klienttekniker

Nedan kan du läsa om olika klienttekniker som används vid utveckling utav hemsidor.

HTML
CSS

HTML

HyperText Markup Language eller HTML en av webbens grundstenar som tillsammans med Cascading Style Sheets och Javascript.

HTML ger strukturen för alla hemsidor som finns på internet. Den står för hur rubriker och stycken delas upp, för att skilja på innehåll och bifoga bilder. Språket är egentligen inte ett programmeringsspråk utan ett märkspråk som har i uppgift att enbart bidra med struktur. Avsaknaden av att inte kunna utföra kommandon bidrar till denna definition.

Användningen utav HTML är allt från att särskilja på rubriker, funktionaliteter som knappar och bilder. Allt detta måste märkas ut för att sedan definieras i antingen CSS eller Javascript. Sökmotorer använder också strukturen för att söka efter relaterbara ord som måste bemärkas i HTML.

Ett enkelt sätt att dela upp sin hemsida är att använda sig först utav rubriker och paragrafer. För att sedan kunna dela upp informationen kan man använda sig utav div som är en tagg för att tilldela ett eller flera stycken en egen sektion. Anledningen till detta är för att man själv vill få bättre struktur alternativt för att ge vissa delar utav hemsidan en specifik design.

CSS

Cascading Style Sheets är den andra grundstenen till en webbplats. Via CSS sköter man designval utav en hemsida och kan ses som ansiktet utåt. Med hjälp utav CSS man ändra bland annat ändra typsnitt, tillsätta färger och former samt infoga animeringar.

CSS kan även ge uttryck åt hur hemsidan skall agera ifall man för musen över innehåll i texten, bilder eller knappar.

Man kan direkt i ett HTML dokument ändra utseende och ge egenskaper åt former och bilder med hjälp utav Style taggen. Detta är dock inget att rekommendera då HTML-koden kan ha en tendens att se stökig ut och svår att underhålla ifall man vill ändra på större delar utav en hemsida. Istället kan man bruka CSS koden i ett separat dokument för att sedan hänvisa dit från HTML dokumentet och på så vis kunna specificera var och vad man vill ge ett stycke eller sektion för design.

Det finns flertalet lösningar på hur objekt skall placeras som till exempel ”flexbox” och ”grid” vilka är verktyg som underlättat för positionering.

JavaScript

Javascript bidrar till funktionaliteten på hemsidor. Enklare animationer kan man utföra med hjälp utav CSS men oftast används javascript när något händer på en hemsida. Exempel på detta kan vara när man skall fylla i formulär eller klicka på knappar som ger gensvar. Svaren kan vara felmeddelanden eller att någonting rör sig på hemsidan.

Många tror att Java och Javascript är samma sak men detta stämmer inte, båda må vara programmeringsspråk men de skiljer sig ifrån varandra i hur de används.

Vanligaste användningsområdet för javascript är att bestämma vad som skall ske när en användare interagerar med en webbplats. Via HTML anropas loopar eller if-satser för att utföra uppgifter på hemsidan. Ett annat vanligt användningsområde är eventlyssnare som lyssnar efter vad användare gör på en hemsida så som att skrolla, röra musen eller interaktioner med hemsidan.
